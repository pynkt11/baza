В Великобритании запустили грандиозную Лотерею для сбора денег на открытие Олимпийских Игр 2012 года

В https://t.me/Lev_games_casinO Великобритании запустили грандиозную Лотерею для сбора денег на открытие Олимпийских Игр 2012 года



Чтобы увеличить финансирование грядущих Олимпийских Игр 2012 года в городе Лондон, власти Великобритании приняли решение о запуске грандиозной лотереи, которая, по их мнению, принесет доход примерно в один миллион фунтов стерлингов. Как считает министр культуры Тессы Джауэлл, эта лотерея необходима для зачинщиков этого грандиозного события, чтобы уменьшить часть расходов при организации Олимпиады.
